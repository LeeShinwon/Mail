### 주요 구현점

- 기본 UI
- 구글 로그인, 로그아웃
- 파이어베이스 연동
- 메일 작성
- 메일 리스트
- 메일 상세보기
- 임시저장된 메일 수정
- 메일 삭제          


### Getting Started

1. Android Studio, Dart를 다운로드해주세요.
2. 새 플러터 프로젝트를 만들어주세요.
3. firebase 에 새 프로젝트를 생성해주세요. (app도 추가)
4. branch별로 나눈 단계를 따라 코딩해주세요.
